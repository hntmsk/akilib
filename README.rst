0.Introduction
-------------------------------------------------------------------------------------------------------------

akilibプロジェクトは、秋葉原で購入出来る部品をライブラリ化し、次世代であるシングルコンピュータで手軽に取り扱えるようにするプロジェクトである。
現在は、Pythonが主であり、サブでC言語を取り入れているが、皆様のご支援によりRubyやGoに移植されることを願っています。
